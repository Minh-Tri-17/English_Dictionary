Quick Formula Memory Mnemonics

  • Continuous (Tiếp diễn): Must contain helper BE + V-ing.
  • Perfect (Hoàn thành): Must contain helper HAVE + V3/pp (Past Participle).
  • Future (Tương lai): Always has modal helper WILL in front of base verbs.

Passive Voice (Câu Bị Động)

General Rule: Active S + V + O → Passive S + BE + V3/pp (+ by O)

Reported Speech (Câu Tường Thuật)

When reporting what someone said, shift tenses backward, adjust pronouns, and update time expressions.

      Statements (Câu kể) S + said (that) + S + V (backshifted)
      Yes/No Questions S + asked (O) + if / whether + S + V (backshifted)
      Wh- Questions S + asked (O) + Wh-word + S + V (backshifted)

      Relative Clauses (Mệnh Đề Quan Hệ)

      Adjectival clauses modifying nouns. Utilizes relative pronouns depending on grammatical roles.

      WHO Subject (People)
      ... N (people) + WHO + V
      WHOM Object (People)
      ... N (people) + WHOM + S + V
      WHICH Subject/Object (Things)
      ... N (thing) + WHICH + V or S+V
      THAT Special Case

      Can replace who, whom, which in defining clauses. Mandatory after:

      • Superlatives (the best, the tallest)
      • Adjectives: only, first, last
      • Indefinite Pronouns (everything, something, all)

      Comparisons (Cấu Trúc So Sánh)

      Formulas vary depending on syllable lengths of adjectives/adverbs (Short vs. Long).

      Short Adjectives / Adverbs

      Comparative (Hơn) adj-er + than
      Superlative (Nhất) the + adj-est

      Long Adjectives / Adverbs

      Comparative (Hơn) more + adj + than
      Superlative (Nhất) the + most + adj

      Symbols & Legend (Ký hiệu viết tắt)

      Additional Auxiliary Structures

      So ... that / Such ... that

      • So + Adj/Adv + that + clause (Quá... đến nỗi mà)
      • Such + a/an + Adj + N + that + clause

      Too ... to V / Enough ... to V

      • Too + Adj/Adv + (for O) + to-V (Quá... để làm gì)
      • Adj/Adv + enough + (for O) + to-V (Đủ... để làm gì)

      Concessions (Sự nhượng bộ)

      Although / Even though / Though + Clause → followed by main clause.

      Gerund (V-ing) Followers

      Gerunds are mandatory after verbs like: avoid, enjoy, finish, keep, mind, suggest, can't help, look forward to.
